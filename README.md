# OMS

Office Management System (OMS) is a Laravel-based web application designed to manage office operations efficiently.

## Features

- Lead Management
- Campaign Management
- AI-powered Email Generation
- Email Marketing
- Employee Management
- Reporting and Analytics

## Description

OMS helps streamline office workflows and business processes through multiple modules. It provides tools for managing leads, running campaigns, generating AI-based emails, handling employee records, and viewing detailed reports for each module.

## Tech Stack

- Laravel
- PHP
- MySQL
- Bootstrap / Frontend UI
