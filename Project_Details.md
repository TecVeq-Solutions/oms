# Office Suite - Project Details

## рЯУБ Folder Structure
The project follows a standard Laravel directory structure:
- `app/`: Contains the core application code (Models, Controllers, Middleware, etc.).
- `bootstrap/`: Bootstrapping scripts.
- `config/`: Application configuration files.
- `database/`: Database migrations, seeders, and factories.
- `public/`: Publicly accessible files (index.php, CSS, JS, images).
- `resources/`: Uncompiled assets like views (Blade/React/Vue), CSS, and JS.
- `routes/`: Application route definitions (web.php, api.php, etc.).
- `storage/`: Compiled Blade templates, file-based sessions, file caches, and other files generated by the framework.
- `tests/`: Automated tests.
- `vendor/`: Composer dependencies.

## рЯІ© Modules Overview
Based on the controllers and models, the application consists of the following key modules:
1. **Authentication & Authorization**: Manages user logins, registration, password resets, and role/permission management (`UserRoleController`, `Auth/*`).
2. **Employee Management**: Handles employee profiles, personal details, bank accounts, and salary payments (`EmployeeController`, `EmployeePersonalDetailController`, etc.).
3. **Attendance & Shift Management**: Tracks employee check-ins/check-outs, records IPs and locations, handles overnight shifts and grace periods (`AttendanceController`, `ShiftController`).
4. **Leave Management**: Manages leave types and leave requests processing (`LeaveRequestController`, `LeaveTypeController`).
5. **Project & Task Management**: A complete task tracking system involving workspaces, projects, task assignments, time logs, comments (with mentions), file attachments, and extension requests (`ProjectController`, `TaskController`, `TaskTimerController`, etc.).
6. **CRM / Lead Management**: Tracks potential clients or leads, and internal notes regarding them (`LeadController`, `LeadNoteController`).
7. **Email Campaigns**: Handles bulk emailing, tracks sent/failed counts, and records email logs (`EmailCampaignController`).
8. **AI Integrations**: Includes AI-powered features/generation using standard models (`AIController`, `AIGenerationController`).
9. **Settings & Configurations**: Office-wide settings, allowed IP addresses, and app notifications (`OfficeSettingController`, `AllowedIpController`).
